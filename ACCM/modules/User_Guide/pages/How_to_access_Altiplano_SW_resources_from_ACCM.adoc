= How to access Altiplano SW resources from {app-short-name}

== Introduction

This section provides the steps required to access direct hyperlinks pointing to the resources inside Altiplano from the
CCM interface.

This allows the operator to seamlessly shift context into the underlying management platform to
troubleshoot localized issues.


[NOTE]
====
Before accessing resources, you must be authenticated to the {app-short-name} as an altiplano user.
Refer to xref:User_Guide:How_to_manage_ACCM_access.adoc[How to Manage {app-short-name} access] section for more details.
====

+
